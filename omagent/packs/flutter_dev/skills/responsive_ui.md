# Responsive UI for Flutter

When building responsive Flutter UI:

1. Use LayoutBuilder and MediaQuery for responsive layouts
2. Use Flex, Expanded, and Flexible for proportional sizing
3. Use Platform.isIOS / Platform.isAndroid for platform-specific widgets
4. Use adaptive widgets: Switch.adaptive, Slider.adaptive
5. Handle safe areas with SafeArea widget
6. Support both portrait and landscape orientations
7. Use ConstrainedBox for min/max sizes
8. Test on multiple screen sizes with Device Preview
