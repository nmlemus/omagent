# ML Modeling Workflow

When asked to build a predictive model:

1. Profile the dataset using `dataset_profile`
2. In Jupyter: clean data, handle missing values, encode categoricals
3. Use `model_train` to train with cross-validation
4. Interpret results: check metrics, feature importances
5. If accuracy is low, try different model types or feature engineering
6. Generate a summary report with visualizations
