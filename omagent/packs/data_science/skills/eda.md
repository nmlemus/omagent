# Exploratory Data Analysis (EDA)

When asked to perform EDA on a dataset:

1. Profile the dataset using `dataset_profile` tool
2. In the Jupyter kernel:
   - Import pandas, matplotlib, seaborn
   - Check data types and convert as needed
   - Analyze missing values pattern
   - Compute correlation matrix for numeric columns
   - Create distribution plots for key numeric columns
   - Create bar charts for categorical columns (if < 20 categories)
   - Identify outliers using IQR method
3. Summarize findings with key insights
4. Suggest next steps for analysis or modeling
